# Playwright_AI

Small Playwright example project.

## Contents

- `01_Basics/test.js` — sample test file.

## Prerequisites

- Node.js (16+)
- npm or yarn

## Install

```bash
cd D:\Development\Playwright_AI
npm install -D playwright
# or: npx playwright install
```

## Run tests

```bash
npx playwright test
```

## Notes

- This repository was initialized locally and contains a `.gitignore`.
- Pushes to the remote may require configuring Git credentials or a Personal Access Token.
